On the Subject of Simon's Dead

Sad that Simon died? You're going to change your mind after solving this module!

See Appendix A for indicator identification reference.
See Appendix B for battery identification reference.
See Appendix C for port identification reference.

  • There are eight buttons on this module, and there are 12 stages.
  • YOU have to determine the colors that are flashing.
  • To defuse the module, see below for "seed instructions"

Seed Instructions:

Step 1
• Start with the bomb's serial number.
• If the bomb has 2 or more PS/2 ports, multiply every numeric value on the serial number by 2. Otherwise, add 5 to every numeric value on the serial number
• If a number goes past 9, subtract 10.
Step 2
• For each indicator on the bomb that has a matching letter on the serial number (x), if the indicator is lit, add x to the letter of the serial number. If the indicator is unlit, subtract x to the letter of the serial number. (See the table below for help)
• Otherwise, for each port name that has a matching letter on the serial number (y), multiply y to the letter. (See the table below for help)
• Otherwise, if there's one or more empty port plates, add 1 to every letter on the serial number. (See the table below for help)
• Otherwise, subtract 1 to every letter on the serial number. (See the table below for help)
Step 3
• Convert every letter to a number. This is now your base seed. To get your final seed, follow the rules below.
• If a value on the seed matches a value (numeric or lettered) on the original serial number, add the digits of the seed, and that number is n. Otherwise, n is now the first digit of the seed. (see "defining n" for help.)
• If a value on the seed is equal to the number of indicators, n is the number of indicators. If there are more lit than unlit indicators, multiply n by 2. Otherwise, if there are more unlit than lit indicators, subtract n by 2. Otherwise, if there are equal lit and unlit indicators, leave n as is. If there are no indicators, n is 5. If n is greater than 9, or less than 0, add or subtract 10 until n is 0-9.
• If a value on the seed is equal to the number of ports, n is the number of port plates. If there's an empty port plate, n is multiplied by 4, and use the least significant digit. Otherwise, if there are no port plates, n is the last digit of the seed.
• n is equal to the number of modules on the bomb, excluding needy modules.
• n is equal to the last digit of the seed squared. If n is one digit, add a 0 to the beginning. Instead of one digit this time, n will be 2 digits.

Now, your seed should be a 12 digit number. See the "result instructions".

Alphabet Math

Every letter in the alphabet has an assigned number.
The letter A is equal to 1, and the letter Z is equal to 26.
If your calculated number is lower than 1, or greater than 26, add or subtract 26 until you get a number 1-26.
Use the least significant digit for doing the steps above. I have provided the letters and numbers for you:

A = 1 | B = 2 | C = 3 | D = 4 | E = 5 | F = 6 | G = 7 | H = 8 | I = 9 | J = 10 | K = 11 | L = 12 | M = 13 | N = 14 | O = 15 | P = 16 | Q = 17 | R = 18 | S = 19 | T = 20 | U = 21 | V = 22 | W = 23 | X = 24 | Y = 25 | Z = 26 |

Defining "n"

"n" is the last position of the seed plus 1. If my seed was 756432, n would be 7, because there are 6 positions, and n is the 7th.
When n is assgined a new number, it's not the position, it's the label for that position.
For example: My seed is 231345, and n is 6, my new seed is 2313456. Value n is 7, so my new seed is 23134567, and so on.

Result instructions

Congratulations, you have your seed!
s will be the position number for the seed.
For the first digit, s is 1, and for the last digit, s is 12.
Multiply the sth position by s, and use the least significant digit.
To find what buttons to press, convert the numbers to letters using the chart in "Alphabet Math"
Starting with one letter at a time, use the chart below to determine what new button to press.
Just like the original Simon Says, you have to input the button along with the buttons from the previous stages.
